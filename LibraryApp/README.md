# 📚 Library App — Sistema de Gestión de Biblioteca

Sistema de consola desarrollado en C# (.NET 8) para gestionar libros, usuarios y préstamos de una biblioteca.

## 🚀 Tecnologías
- C# 12 / .NET 8
- Consola interactiva con colores y emojis
- Arquitectura por capas: Models / Services / UI
- Control de versiones: Git Flow

## 📦 Estructura del proyecto
\```
LibraryApp/
├── Models/          → Entidades del dominio
├── Services/        → Lógica de negocio con List<T>
├── UI/              → Menús y presentación
└── Program.cs       → Punto de entrada
\```

## ▶️ Cómo ejecutar
\```bash
cd LibraryApp
dotnet run
\```

## 🌿 Ramas Git Flow
- `main` → producción estable
- `develop` → integración de features
- `feature/*` → desarrollo de módulos
